# KindleEar上传脚本

打开 Google 云端 Shell，复制粘贴以下命令，按照脚本的提示输入 Gmail 和 APPID 等信息即可轻松上传。

```
rm -f uploader.sh* && \
wget https://github.com/yuyzhen/KindleEar-Uploader/blob/master/uploader.sh && \
chmod +x uploader.sh && \
./uploader.sh
```
